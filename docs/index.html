What it does

Adds a small page-edge drop zone for explicit drag and drop actions.

  • Search selected text.
  • Open dropped URLs and links.
  • Use the browser default search provider when supported.

Controls

Configure the workflow from the extension options page.

  • Top edge, bottom edge, or both.
  • Current tab or new tab behavior.
  • Fallback search URL template.

Privacy

No analytics, tracking, ads, or remote code.

  • Processes only content you explicitly drop.
  • Stores only extension preferences.
  • Open source under MIT.
