Privacy Policy

Privacy

Vivaldi Drag Search does not collect, sell, share, or transmit personal data to the extension author.

Summary

The extension processes dropped text or URLs only after the user explicitly drops content on the page-edge drop zone.

Dropped Content

URL drops are opened in the current tab or a new tab. Text drops are sent to the browser's configured default search provider through the Chromium search API, or to the user-configured fallback search URL if that API is unavailable.

Stored Data

The extension stores only user preferences such as drop-zone position, search mode, and tab-opening behavior using chrome.storage.sync.

No Tracking

The extension does not use analytics, advertising, tracking, remote code, or third-party extension services.

The use of information received from browser APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.
