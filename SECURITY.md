# Security Policy

## Reporting a Vulnerability

Please report security issues privately to the project maintainer before opening a public issue.

If the project is hosted on GitHub, use one of these options:

- Open a private security advisory if available.
- Email the maintainer address listed on the repository profile.

Please include:

- A clear description of the issue.
- Steps to reproduce.
- Browser and extension version.
- Any relevant screenshots or logs.

## Security Design

Vivaldi Drag Search does not use remote code, analytics, advertising, or tracking. It processes dropped text or URLs only after the user explicitly drops content on the extension drop zone.
